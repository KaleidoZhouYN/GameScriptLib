# 概述

GameScritLib是一个个人使用的游戏脚本库，主要想要实现的功能有：

1. 截图，包括后台截图（OPENGL、DirectX渲染），前台截图
2. Windows消息模拟
3. 找图
4. 文本检测+文字识别+翻译
5. 特殊目标检测，AI脚本机制
6. AI强化学习
7. ...
